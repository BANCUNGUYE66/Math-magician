- Node
- ESlint set up

### Setup

Clone this repository to your desired folder:
<br>
<br>gh repo clonehttps://github.com/BANCUNGUYE66/Math-magician



### Install

Install this project with:



```sh
  
  gh repo clone BANCUNGUYE66/Math-magician
  cd Math_magician
  npm install -y
```
